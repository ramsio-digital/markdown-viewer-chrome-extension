# Contributing

Thank you for your interest in RAMSIO Markdown Viewer!

This repository serves as the **public issue tracker** for the RAMSIO Markdown Viewer Chrome Extension. The source code is maintained in a private repository.

## How to Contribute

### Report a Bug

1. Check [existing issues](../../issues) to avoid duplicates
2. Open a [bug report](../../issues/new?template=bug_report.md) with details
3. Include your Chrome version, extension version, and OS
4. Add screenshots if possible

### Request a Feature

1. Check [existing feature requests](../../issues?q=label%3Aenhancement) to avoid duplicates
2. Open a [feature request](../../issues/new?template=feature_request.md)
3. Describe the problem and your proposed solution

### General Questions

For general questions or support, please:
- Visit our [Support page](https://www.ramsio.tools/support)
- Email [support@ramsio.tools](mailto:support@ramsio.tools)

## Response Time

We aim to respond to all issues within 48 hours on business days.

## Code of Conduct

Be respectful and constructive. We're building tools to help developers — let's keep the conversation productive.
